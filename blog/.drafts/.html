Hello. This is a quick reference on Python environment variables.

The main idea is that the os module provides a dictionary called environ, which works the way you would expect.

import os

// get a variable
config = os.environ.get("XDG_CONFIG_HOME")
// or
config = os.environ["XDG_CONFIG_HOME"]

// Won't persist after the script is finished
os.environ["XDG_CONFIG_HOME"] = "/home/foo/.bar"
print(os.environ["XDG_CONFIG_HOME"])

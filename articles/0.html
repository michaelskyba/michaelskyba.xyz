Block a site using /etc/hosts

Simply add 0.0.0.0 mysite to your /etc/hosts file. Example: 0.0.0.0 facebook.com

Here, facebook.com will basically be an alias for "0.0.0.0", which acts as a dummy address.

Note that this isn't completely reliable. If it's not working, try adding a second entry with "www." and also try restarting your browser.
